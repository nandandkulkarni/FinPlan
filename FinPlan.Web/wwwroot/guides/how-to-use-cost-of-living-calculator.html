Cost of Living Planner
Quick guide — capture today’s expenses and project retirement costs

This guide explains the planner UI, steps to create plans, adjust inflation, and interpret totals — fast and practical.

Overview

The Cost of Living Planner lets you enter categories and items (subcategories), set how each item scales into retirement, and project totals using a global or per-item inflation rate. Changes are auto-saved but can be undone shortly after removal.

Quick Start
1
Open the planner
Visit /cost-of-living. The page loads your saved plans (Plan-A by default). Use the tabs to switch or create new plans.
2
Add a category
Click Add Category, enter a name, and the planner will add a starter item for that category.
3
Add items
Use the green Add Item button in a category header to add subcategory rows. Click the trash icon to remove a single row.
4
Edit item details
Click View/Edit to open the item modal. Change frequency, current value, retirement adjustment, or per-item inflation.
5
Set years & inflation
Use the controls in the top strip: Years to Retirement and Inflation Rate (%). Changing inflation immediately recalculates totals.
Understanding retirement adjustments
  • Same — item amount remains identical in retirement.
  • Adjust for Inflation — item is inflated using either the global inflation rate or a per-item custom percent.
  • Custom % — retirement value is a percentage of current value.
  • Manual — you specify the retirement monthly value directly.
Totals & live updates

Totals (Current & Retirement) appear in the totals strip. When you change the global inflation rate or a per-item inflation percent, the retirement totals update immediately. The planner debounces changes and autosaves — a small spinner indicates saving.

Shortcuts & tips
  • Keyboard: while editing a header, press Enter to save or Escape to cancel.
  • Undo: after removing an item you'll see an Undo toast for a few seconds.
  • Tabs: use the tab controls to create additional plans (Plan-B, Plan-C), each saved independently.
  • Performance: large plans autosave in background — no blocking UI.
FAQ
Can I set different inflation for an item?
Yes — open the item modal, choose Adjust for Inflation, set Inflation Source to Custom, and enter a percent.
How are frequencies handled?
The planner converts weekly, yearly etc. into a monthly-equivalent before applying retirement adjustments.
Where are plans saved?
Plans are saved for your user GUID and are retrievable via the tab list. Use the Delete All button to remove saved plans (confirmation required).
Contact & support

If you find issues or want improvements, open an issue or contact the developer team. For quick debugging enable the app’s debug console (if available) or copy the last save timestamp displayed on the planner UI.
